# angularjs-testing

## Pré-requis

* Avoir [nodejs](http://nodejs.org/) et npm
* Avoir [bower](http://bower.io) (npm install -g bower)

## Installation des dépendances

(aka "Téléchargement d'internet")

```
$ npm install
$ bower install
```
